## 1. 전체 개요

### 시스템 구성 (5개 모듈)

```
┌──────────────────────────────────────┐
│  🏠 대시보드                          │
│   (재직 인원 / 오늘 출근 / 휴가 대기  │
│    / 이번달 급여 요약)                │
├──────────────────────────────────────┤
│  👤 인사 관리   (Employee Master)     │
│   사번 / 이름 / 부서 / 직급 / 입사일  │
├──────────────────────────────────────┤
│  ⏱ 근태 관리   (Attendance)           │
│   ⭐ 출근 / 퇴근 자동 기록 + 시계 연동│
│   근무시간 자동 계산 (점심 1h 차감)   │
├──────────────────────────────────────┤
│  🌴 휴가 관리   (Leaves)              │
│   연차 / 병가 / 휴직 신청 + 승인      │
├──────────────────────────────────────┤
│  💰 급여 관리   (Payroll)             │
│   ⭐ 근태 자동 연계 → 자동 산출       │
│   기본급 + 초과수당 - 공제 = 실수령액 │
└──────────────────────────────────────┘
```

### 시작 화면
- 👥 **서진정밀 인사/근태 시스템** 인트로 → **Enter** 또는 클릭
- 메인 화면: 좌측 사이드바 5개 메뉴

## 3. STEP 1 — 인사 등록

### 👤 인사 관리 메뉴 클릭

기본 사원이 5명 등록되어 있음. 새 사원 1명 추가 예시:

| 항목 | 입력값 |
|---|---|
| 사번 * | `E2026006` |
| 이름 * | `한지민` |
| 부서 | `품질팀` (드롭다운: 생산팀/품질팀/경영지원/영업팀/연구개발/품질관리) |
| 직급 | `사원` (드롭다운: 사원/대리/과장/차장/부장/이사) |
| 입사일 | `2026-05-01` |
| 연락처 | `010-6666-7777` |
| 이메일 | `han@seojin.kr` |
| 재직상태 | `재직` |
| 기본급(원) | `2900000` |

→ **[사원 등록]** (녹색)

### 수정 / 삭제
- **수정**: 목록에서 사원 행 클릭 → 폼 자동 채워짐 → 값 변경 → **[사원 수정]**
- **삭제**: 행 클릭 → **[사원 삭제]** (소프트 삭제, active=0)

### 행 색상
- ⚪ 재직: 일반
- 🟠 휴직: 주황
- 🔴 퇴사: 빨강

## 4. STEP 2 — 근태 자동 기록 (출퇴근)

### ⏱ 근태 관리 메뉴 클릭

### 4.1 자동 출퇴근 패널 (화면 상단)

**5월 2일 아침 09:00**: 김철수 출근
1. 사원 드롭다운: `E2026001 김철수` 선택
2. 상태: **⚪ 미출근** 표시됨
3. **[🟢 출근]** 버튼 클릭
4. → 알림: "✅ 김철수 님 / 출근 시각: 09:00 / 상태: 정상"
5. 상태 변경: **🟢 출근 09:00 (근무중)**

**5월 2일 저녁 18:00**: 김철수 퇴근
1. 사원: `E2026001 김철수` 선택 (또는 그대로)
2. **[🔴 퇴근]** 버튼 클릭
3. → 알림: "✅ 퇴근 시각: 18:00 / 근무 8h / 초과 0h"
4. 상태: **✅ 출근 09:00 → 퇴근 18:00**

### 4.2 야근 케이스 (5월 5일)

| 시간 | 동작 |
|---|---|
| 09:00 | [🟢 출근] → 정상 |
| 21:30 | [🔴 퇴근] → 근무 8h + **초과 3.5h** (점심 1h 차감 후) |

### 4.3 지각 케이스 (5월 8일)

| 시간 | 동작 |
|---|---|
| 09:30 | [🟢 출근] → 자동 **지각** 분류 |
| 18:00 | [🔴 퇴근] → 근무 8h |

### 4.4 수동 등록 (잊어버린 어제 분)

**자동 폼 (특별 케이스용)**

| 항목 | 입력값 |
|---|---|
| 사원 * | `E2026001 김철수` |
| 근무일 * | `2026-05-01` (자동 = 오늘, 수정 가능) |
| 출근 (HH:MM) | `09:00` |
| 퇴근 (HH:MM) | `18:00` |
| 상태 | `정상` (드롭다운: 정상/지각/조퇴/결근/반차/출장) |
| 비고 | `시스템 등록 누락` |

→ **[근태 등록]** (녹색)

### 4.5 5월 한 달치 시뮬레이션 (요약)

| 일자 | 출근 | 퇴근 | 근무 | 초과 | 상태 |
|---|---|---|---|---|---|
| 5/2 (금) | 09:00 | 18:00 | 8h | 0 | 정상 |
| 5/5 (월) | 09:00 | 21:30 | 8h | 3.5h | 정상 |
| 5/6 (화) | 09:00 | 19:00 | 8h | 1h | 정상 |
| 5/7 (수) | 09:00 | 20:00 | 8h | 2h | 정상 |
| 5/8 (목) | 09:30 | 18:00 | 8h | 0 | **지각** |
| 5/9 (금) | 09:00 | 19:30 | 8h | 1.5h | 정상 |
| 5/12 (월) | (휴가) | - | - | - | (연차) |
| 5/13~5/30 | 09:00 | 18:00 ~ 19:00 | ... | ~4h 추가 | 정상 |

→ **합계: 근무일 20일 / 총 근무 160h / 초과 12h / 지각 1회 / 연차 1일**

## 5. STEP 3 — 휴가 신청 / 승인

### 🌴 휴가 관리 메뉴 클릭

### 5.1 김철수 연차 신청 (5월 12일)

| 항목 | 입력값 |
|---|---|
| 사원 * | `E2026001 김철수` |
| 구분 * | `연차` (드롭다운: 연차/반차/병가/경조사/휴직/기타) |
| 시작일 * | `2026-05-12` |
| 종료일 * | `2026-05-12` |
| 사유 | `개인사유 (자녀 학교 행사)` |
| 승인상태 | `대기` |

→ **[휴가 신청]** (녹색)
→ 일수 자동 계산: **1일**

### 5.2 승인 처리 (관리자가 검토 후)

1. 목록에서 김철수 연차 행 클릭 → 폼 자동 채워짐
2. **[✓ 승인]** 버튼 클릭 (파란색)
3. → "휴가가 승인되었습니다."
4. 상태 변경: 대기 → **승인** (행 색상 녹색)

**이영희 반차** (오후 외출)

|---|---|

| 구분 | `반차` |
| 시작/종료 | `2026-05-15` |
| 사유 | `병원 진료` |

**박민수 병가** (3일)
| 항목 | 값 |
|---|---|
| 사원 | `E2026003 박민수` |
| 구분 | `병가` |
| 시작 | `2026-05-20` |
| 종료 | `2026-05-22` |
| 사유 | `독감` |

→ 일수 자동: **3일**

## 6. STEP 4 — 급여 자동 계산

### 💰 급여 관리 메뉴 클릭

### 6.1 김철수 5월 급여 자동 산출

1. **사원** 드롭다운: `E2026001 김철수` 선택
   → 인사정보에서 **기본급 4,200,000 자동 로드**
2. **급여월**: `2026-05` 입력
   → **즉시 5월 근태 자동 집계** + 모든 항목 자동 계산

### 6.2 자동 표시되는 정보

```
📊 2026-05 근태 자동 집계 — 근무일 20일 / 총 160.0h / 초과 12.0h
   결근 0일, 지각 1회, 반차 0회

💰 자동 산출 — 시급 20,096원 / 초과수당 361,728원
   보험 394,800원 + 세금 126,000원
   결근차감 0원 + 지각차감 10,048원
   = 공제 530,848원

실수령액: 4,030,880원
```

### 6.3 자동 계산식 상세

| 항목 | 계산식 | 값 |
|---|---|---|
| **시급** | 기본급 / 209h | 20,096원 |
| **초과수당** | 12h × 시급 × 1.5 | 361,728원 |
| **결근차감** | 일급 × 0일 | 0원 |
| **지각차감** | 시급 × 0.5 × 1회 | 10,048원 |
| **4대보험** | 기본급 × 9.4% | 394,800원 |
| **소득세** | 기본급 × 3% | 126,000원 |
| **총 공제** | 보험+세금+결근+지각 | 530,848원 |
| **실수령액** | 기본급+초과수당-공제 | **4,030,880원** |

### 6.4 폼 자동 입력 결과

| 항목 | 자동 입력값 | 수동 조정 가능 |
|---|---|---|
| 기본급(원) | `4,200,000` | ✓ |
| 초과수당 | `361,728` | ✓ |
| 상여금 | `0` | ✓ (필요시 입력) |
| 공제 | `530,848` | ✓ |
| 지급일 | `2026-05-25` (또는 직접 입력) | ✓ |

→ **[급여 등록]** (녹색)

## 8. 업무 흐름 다이어그램

```
[인사 등록]
👤 사원 등록 (사번/이름/부서/기본급)
       ↓
[일일 근태]
⏱ 출근 자동 기록 (현재 시각)
       ↓
   (점심 / 업무)
       ↓
⏱ 퇴근 자동 기록 (현재 시각)
       ↓
   근무시간 자동 계산
   (점심 1h 차감, 8h 초과 = 야간)
       ↓
[휴가 발생 시]
🌴 휴가 신청 → 승인 대기 → 관리자 [✓ 승인]
       ↓
[월말 급여]
💰 사원 + 월 선택
       ↓
   인사관리 → 기본급 자동 로드
   근태관리 → 근무일/초과시간/지각/결근 집계
       ↓
   자동 산출:
   - 시급 = 기본급 / 209h
   - 초과수당 = 초과h × 시급 × 1.5
   - 4대보험 = 기본급 × 9.4%
   - 소득세 = 기본급 × 3%
   - 실수령액 = 기본급 + 초과수당 + 상여 - 공제
       ↓
   💰 급여 등록
       ↓
[월별 확인]
🏠 대시보드 → 이번달 급여 합계 표시
```

## 9. 자주 쓰는 단축 팁

### 화면 조작
| 단축키 | 동작 |
|---|---|
| Enter (인트로) | 메인 진입 |
| 마우스 휠 | 페이지 수직 스크롤 |
| Cmd/Ctrl + Tab | 앱 전환 |

### 데이터 조작
| 동작 | 방법 |
|---|---|
| 신규 등록 | 폼 입력 → 녹색 [등록] 버튼 |
| 수정 | 목록 행 클릭 → 폼 자동 채워짐 → 주황 [수정] |
| 삭제 | 행 클릭 → 빨강 [삭제] → 확인 |
| 출근 | 사원 선택 → 🟢 [출근] (시각 자동) |
| 퇴근 | 사원 선택 → 🔴 [퇴근] (시간 자동 계산) |
| 휴가 승인 | 행 클릭 → 🔵 [✓ 승인] |
| 급여 자동 계산 | 사원 + 월 선택 → 자동 |

### 색상 의미

**상태 색상 (행 배경)**
- 🟢 정상 / 합격 / 승인 / 정상 출근 — 녹색
- 🟠 지각 / 반차 / 대기 / 정비 — 주황
- 🔴 결근 / 불합격 / 반려 / 퇴사 — 빨강
- ⚪ 일반 / 휴직 / 미입력 — 회색

**버튼 색상**
- 🟢 녹색 = 등록 / 저장
- 🟠 주황 = 수정
- 🔴 빨강 = 삭제
- 🔵 파랑 = 액션 (출근/조회/승인)
