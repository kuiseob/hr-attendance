# SEOJIN HR & ATTENDANCE - 인사/근태 관리 시스템

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
![Python](https://img.shields.io/badge/python-3.9+-green.svg)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-blue.svg)

중소기업 및 제조업 현장을 위한 통합 인사/근태 관리 시스템입니다.

## 📋 주요 기능

### 1. **직원 관리** 👥
- 직원 정보 등록 및 관리
- 부서, 직급, 연락처 관리
- 입사일, 기본급 설정

### 2. **근태 관리** ⏰
- 자동 출퇴근 기록
- 스마트 시간 계산
  - **정상근무**: 08:30 ~ 17:30 (1시간 휴게) = 8시간
  - **초과근무**: 17:30 ~ 22:00 (0.5시간 휴게)
  - **야간근무**: 22:00 ~ 05:30 (휴게 없음)
  - **특근**: 주말/공휴일 시간 + 야간특근

### 3. **휴가 관리** 📅
- 휴가 신청 및 승인 시스템
- 휴가 유형별 관리 (연차, 병가, 특휴, 무급)
- 휴가 일수 자동 계산

### 4. **급여 관리** 💰
- 월별 급여 자동 계산
- 시간당 임금 설정
- 보너스 및 공제액 관리
- 급여 내역서 생성

### 5. **통계 및 분석** 📊
- 월별 근무 현황 통계
- 직원별 근무 시간 분석
- 초과근무 현황 조회

상세한 사용자 매뉴얼은 [`SEOJIN_HR_사용자매뉴얼.pdf`](./SEOJIN_HR_사용자매뉴얼.pdf)를 참고하세요.

### 빠른 시작 가이드

#### 1️⃣ 직원 추가
```
좌측 메뉴 → 직원관리 → 사번, 이름, 부서 입력 → 저장
```

#### 2️⃣ 출퇴근 기록
```
좌측 메뉴 → 근태관리 → [출근] 또는 [퇴근] 버튼 클릭
```

#### 3️⃣ 급여 계산
```
좌측 메뉴 → 급여관리 → 직원 선택 → 월 선택 → 자동 계산
```

## 💾 데이터 저장

모든 데이터는 `hr.db` SQLite 데이터베이스에 저장됩니다:

- **employees**: 직원 정보
- **attendance**: 근태 기록
- **leaves**: 휴가 신청
- **payrolls**: 급여 기록

데이터베이스는 자동으로 생성되며, 프로그램 재설치 후에도 유지됩니다.

## ❓ FAQ

### Q. 데이터베이스 파일 위치는?
**A.** `hr.db`는 프로그램과 같은 디렉토리에 자동 생성됩니다.

### Q. 여러 사용자 동시 접근 가능한가?
**A.** 단일 사용자 또는 순차적 사용 권장. 네트워크 공유 폴더 사용 시 여러 사용자 가능.

### Q. 근무 시간 계산이 잘못되었다면?
**A.** 출퇴근 시간을 다시 확인하고 저장하세요. 시간당 임금 설정도 확인하세요.

### Q. 프로그램이 실행 안 됨?
**A.** 
- Windows: Microsoft C++ 재배포 패키지 설치 확인
- macOS: Python 3.9+ 설치 확인
- 관리자에게 문의
